## Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/LitReview.git
cd LitReview

# Install dependencies
just install

# Start development server
just dev
```

### First Time Setup

1. Launch LitReview Pro
2. Navigate to **API Configuration** in the settings
3. Add your preferred LLM provider API keys
4. Select your model and start generating!
